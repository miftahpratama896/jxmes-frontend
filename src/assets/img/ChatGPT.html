Untuk menambahkan pagination ke tabel Anda dengan batas 250 data per halaman, Anda dapat menggunakan state tambahan untuk melacak halaman saat ini dan jumlah total halaman yang dibutuhkan berdasarkan jumlah total data. Berikut adalah langkah-langkah umum yang dapat Anda ikuti:

  1. Tambahkan state tambahan untuk melacak halaman saat ini.
  2. Hitung jumlah total halaman berdasarkan jumlah total data dan batas data per halaman (250).
  3. Tambahkan logika untuk membagi data ke dalam halaman-halaman yang sesuai dengan halaman saat ini.
  4. Tambahkan tombol atau tautan untuk navigasi antara halaman-halaman.
  5. Perbarui rendering tabel agar hanya menampilkan data untuk halaman saat ini.

Berikut adalah implementasinya:

javascript
import React, { useState, useEffect } from 'react';
import axios from 'axios';

const TableWithPagination = () => {
  const [data, setData] = useState([]);
  const [error, setError] = useState(null);
  const [updating, setUpdating] = useState(false);
  const [currentPage, setCurrentPage] = useState(1);
  const itemsPerPage = 250;

  // Function to fetch data
  const fetchData = async () => {
    try {
      setUpdating(true);
      const response = await axios.post('http://172.16.200.28:3000/scan-jx2-jx', {
        // Your request payload here
      });
      setData(response.data);
    } catch (error) {
      setError('Error fetching data');
    } finally {
      setUpdating(false);
    }
  };

  // UseEffect to fetch data initially and on currentPage change
  useEffect(() => {
    fetchData();
  }, [currentPage]); // eslint-disable-line react-hooks/exhaustive-deps

  // Calculate total pages
  const totalPages = Math.ceil(data.length / itemsPerPage);

  // Handle page change
  const handlePageChange = (pageNumber) => {
    setCurrentPage(pageNumber);
  };

  // Get current items
  const indexOfLastItem = currentPage * itemsPerPage;
  const indexOfFirstItem = indexOfLastItem - itemsPerPage;
  const currentItems = data.slice(indexOfFirstItem, indexOfLastItem);

  return (
    <div>
      <table>
        {/* Table header */}
        {/* Table body with currentItems */}
      </table>

      {/* Pagination */}
      <div>
        {Array.from({ length: totalPages }, (_, index) => (
          <button key={index} onClick={() => handlePageChange(index + 1)}>
            {index + 1}
          </button>
        ))}
      </div>

      {/* Loading indicator */}
      {updating && <div>Loading...</div>}
    </div>
  );
};

export default TableWithPagination;

Selain itu, Anda perlu menambahkan bagian untuk render tabel sesuai dengan data pada currentItems.

Pergantian halaman akan memicu pengambilan data baru dari server, dan tabel akan diperbarui dengan data yang sesuai. Juga, terdapat tombol untuk setiap halaman sehingga pengguna dapat dengan mudah beralih antara halaman-halaman yang berbeda.

Untuk mengekspor tabel ke dalam format Excel di ReactJS, Anda dapat menggunakan pustaka pihak ketiga seperti xlsx. xlsx adalah pustaka JavaScript yang memungkinkan Anda membuat file Excel di browser.

Berikut adalah langkah-langkah umum untuk mengekspor tabel menjadi file Excel di ReactJS menggunakan xlsx:

  1. Instalasi xlsx:

    Anda dapat menginstal pustaka xlsx menggunakan npm atau yarn:

    bash
    npm install xlsx

    atau

    bash
    yarn add xlsx

  2. Import xlsx dan buat fungsi untuk mengekspor tabel:

    javascript
    import React from 'react';
import * as XLSX from 'xlsx';

const exportToExcel = (data) => {
  const fileType = 'application/vnd.openxmlformats-officedocument.spreadsheetml.sheet;charset=UTF-8';
  const fileExtension = '.xlsx';
  const ws = XLSX.utils.json_to_sheet(data);
  const wb = { Sheets: { 'data': ws }, SheetNames: ['data'] };
  const excelBuffer = XLSX.write(wb, { bookType: 'xlsx', type: 'array' });
  const blob = new Blob([excelBuffer], { type: fileType });
  const fileName = 'example' + fileExtension;
  const url = URL.createObjectURL(blob);
  const a = document.createElement('a');
  a.href = url;
  a.download = fileName;
  a.click();
};

const Table = ({ data }) => {
  return (
    <div>
      <table>
        {/* Render your table here */}
      </table>
      <button onClick={() => exportToExcel(data)}>Export to Excel</button>
    </div>
  );
};

export default Table;

  3. Panggil fungsi exportToExcel ketika pengguna mengklik tombol "Export to Excel".
